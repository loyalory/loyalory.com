# Accordion

Accordion is a component to rendering collapsed items.

```js
import Accordion from '@loyalory/common/src/components/Accordion';

<Accordion
  options={[
    {
      title: 'Option 1',
      content: 'Collapsed content',
    },
    {
      title: 'Option 2',
      content: 'Collapsed content',
    },
  ]}
  renderItem={({ option, key, onSelect, isSelected }) => (
    <div onClick={onSelect}>
      {option.title} {isSelected && option.content}
    </div>
  )}
/>;
```
